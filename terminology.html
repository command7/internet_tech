Terminology

Internet

The Internet is a globalized network of interconnected systems through which devices from different geographic locations can communicate and share information with each other using standardized protocols.

Web

The web is a hypertext based information system that contains pages accessible through the internet. These pages are accessible through hypertext references to the location of these pages.

Hypertext

It is a software system that links text to related information, graphics etc., It is accessed usually through point and click method.

Multimedia

Multimedia can be described as the integration of multiple types of media such as audio, video, text, graphics etc., together.

CLI

Command Line Interface(CLI) is a means of interacting with a computer through the terminal and it can be used to perform various operations. However, command line interfaces can only display text based content. In order to operate it, the user needs to know a series of commands.

GUI

Graphical User Interface(GUI) is also a means by which one can interact with a computer. However, in addition to the capability of executing various operations, it can display images, videos and other non text based content also. It makes it much easier for the user to understand and interact with the computer compared to the command line interface.

HCI

Human Computer Interaction(HCI) is a branch of study that deals with interaction between users and computers and how interfaces and designs influence and affect users.

FTP

File Transfer Protocol is a standardized protocol followed by devices connected to the internet to send, receive and modify files in a server or database.

SSH

Secure Socket Shell(SSH)is a secure network protocol that allows users to log into remote systems through an encrypted channel and offers more secure authentication mechanisms.

HTTP

Hyper Text Transfer Protocol(HTTP) is a standardized protocol used to share, request, receive, edit or delete hyper text documents across the web.

Telnet

Telnet is a protocol that allows users to log into remote computers through means of the Internet without the need of physical connections.

HTML

Hyper Text Markup Language(HTML) is a programming language that is used to inform the browser on how to display different content. It is used to specify alignment, formatting, placement etc., that are to be applied to content before presenting to the user. The browser uses these markups to render web page content appropriately.

CSS

Cascading Style Sheets are a type of programming language that are used to format content on the web page in a more organized and efficient manner.
