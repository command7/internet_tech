Terminology

A globalized network of interconnected systems through which devices from different geographic locations can communicate and share information with each other using standardized protocols.
World Wide Web
HyperText
MultiMedia
Hypertext
Command Line Interface is a means of interacting with a computer through the terminal that can be used to perform various operations. However, command line interfaces can only display text based content. In order to operate it, the user needs to know a series of commands.
Graphical User Interface is also a means by which one can interact with a computer. However, in addition to the capability of executing various operations, it can display images, videos and other non text based content also. It makes it much easier for the user to understand and interact with the computer compared to the command line interface.
HCI
File Transfer Protocol is a standardized protocol followed by devices connected to the internet to send, receive and modify files in a server or database.
SSH
Hyper Text Transfer Protocol is a standardized protocol used to share, request, receive, edit or delete hyper text documents across the web.
telnet
Hyper Text Markup Language is a programming language that is used to inform the browser on how to display different content. It is used to specify alignment, formatting, placement etc., that are to be applied to content before presenting to the user. The browser uses these markups to render web page content appropriately.
